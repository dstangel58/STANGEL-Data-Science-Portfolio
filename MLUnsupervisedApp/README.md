🌍 Country Categorization 🌍

## **Project Summary:** 
In this “Country Categorization” project, you can take a variety of datasets with well-being statistics (GDP, health spending, income, trade) from countries all over the world. Using both KMeans and PCA, you can select which features interest you most and reduce them to a two dimensional graph.

## 📖 **Steps to Use:** 📖
* Clone the Repository
    * git clone https://github.com/dstangel58/STANGEL-Data-Science-Portfolio.git
    * cd STANGEL-Data-Science-Portfolio/MLUnsupervisedApp
* Download requirements.txt
    * pip install -r requirements.txt
* Run the code!
    * streamlit run main_code.py

* Deployed App Link: https://dstangel58-stangel-data-scien-mlunsupervisedappmain-code-uiey5f.streamlit.app/
